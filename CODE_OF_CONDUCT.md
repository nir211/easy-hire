# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and abusive behavior are not tolerated. Maintainers may remove, edit, or reject contributions and comments that violate this standard.

To report unacceptable behavior, contact the maintainer via the repository's contact channel (e.g. a private GitHub issue or the email listed on the maintainer's profile). Reports will be handled confidentially.
